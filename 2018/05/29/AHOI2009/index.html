AHOI2009维护序列

发表于 2018-05-29 | 分类于 OI

Description

老师交给小可可一个维护数列的任务，现在小可可希望你来帮他完成。 有长为N的数列，不妨设为a1,a2,…,aN 。有如下三种操作形式： (1)把数列中的一段数全部乘一个值; (2)把数列中的一段数全部加一个值; (3)询问数列中的一段数的和，由于答案可能很大，你只需输出这个数模P的值。

Solution

线段树裸题。但是有一个问题就是这里有一个区间乘的操作，应用之前区间加的解法使用lazytag，给区间乘也打上lazytag，在pushdown时先下传乘法再下传加法，加法的标记需要乘上乘法的标记。
比如1,2,3,4,5,6这个序列，更改[1,6]先+1再×2，先把+1的标记×2变成+2，再下传。
